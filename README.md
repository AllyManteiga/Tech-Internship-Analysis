# Internship Analysis

### [View Interactive Tableau Dashboard Here](https://public.tableau.com/shared/6WP5CY6X2?:display_count=n&:origin=viz_share_link)

#### I built this project to expand my data analytic skills. Utilizing Kaggle, I downloaded a dataset of over 120,000 job listings from 2025. I cleaned the dataset and specifcally analyzed internship postings to identify the most in demand skills, positions, and geographic hubs for students.

#### Tools Used:
* **Python & Jupyter Notebook** (Pandas for cleaning, Matplotlib for initial data analysis)
* **Tableau** (Creating comprehensive visualizations)
