# Internship Analysis

### [View Interactive Tableau Dashboard Here](https://public.tableau.com/shared/6WP5CY6X2?:display_count=n&:origin=viz_share_link)

#### This project analyzed over 1,000 internship postings to identify the most in demand skills, positions, and geographic hubs for students

#### Tools Used:
* **Python** (Pandas for cleaning, Matplotlib for initial data analysis)
* **Tableau** (Creating comprehensive visualizations)
* **Jupyter** Notebooks
